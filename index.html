📌 Instructions for Using the Number Selection System

1️⃣ Enter a number (1-100).

Type a number in the input field and click "Check Availability".

If the number is already taken, an alert will notify you to choose another.

2️⃣ If available, a 30-second countdown starts.

You will see a timer and a new input field for entering your name.

3️⃣ Enter your name and confirm within 30 seconds.

Type your name and click "Confirm" before the timer runs out.

If you don’t confirm within 30 seconds, the number is released.

4️⃣ Approved numbers are displayed in a list.

Once confirmed, your number and name will be added to the list of approved numbers.

5️⃣ Data is saved even after refreshing the page.

Numbers remain reserved after a page reload because they are stored in localStorage.

📌 Note: If a number is already taken, it cannot be selected again.
